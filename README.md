 # 🌦️ Weather App - Real-Time Weather Forecast Platform

Weather App is a simple and responsive web application that provides real-time weather updates for any city worldwide. It allows users to check current weather conditions, temperature, humidity, and more using an external weather API.

 # ✨ Features
👤 For Users

🔍 City-Based Search

Enter any city name to get instant weather details.

🌡️ Real-Time Weather Data

Displays temperature, weather condition, humidity, wind speed, and pressure.

📍 Location-Based Weather (Optional)

Detects user location and shows current weather automatically.

🎨 Clean & Responsive UI

Simple and user-friendly interface built with HTML, CSS, and JavaScript.

🌈 Dynamic Weather Icons

Icons and background change based on weather conditions (Sunny, Rainy, Cloudy, etc.).

⚡ Fast API Integration

Fetches live data using a weather API (like OpenWeather API).

📱 Mobile Friendly

Fully responsive design for mobile, tablet, and desktop.

 # 🚀 Getting Started
🔧 Prerequisites
Web Browser (Chrome, Edge, etc.)
Internet Connection
Weather API Key (e.g., OpenWeather)

 # 📥 Installation
Clone the repository
git clone
cd Weather_app
Open index.html in your browser
OR use Live Server in VS Code.

# 🔑 API Setup
Go to OpenWeather website
Generate your API key
Replace it in script.js:
const apiKey = "your_api_key_here";

# ▶️ How It Works
User enters a city name
JavaScript sends request to weather API
API returns weather data in JSON format
Data is displayed dynamically on UI
